# Hybrid Product Recommendation Engine

Overview:
Developed a machine learning-based recommendation system that suggests relevant products to users based on product features and customer preferences.

Technologies Used:
- Python
- Pandas
- NumPy
- Scikit-learn
- Power BI

Features:
- Product recommendation generation
- Data preprocessing and analysis
- Customer and product insights
- Interactive dashboards

Project Output:
The system recommends similar and relevant products to users and provides insights into customer behavior and product performance through dashboards.
